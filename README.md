# Sushi Dotfiles

This is a rice I made over the course of a few days for [Riceathon](https://riceathon.hackclub.com). It uses a mix of online themes, fully custom themes, and customized themes from other rices.

## Configured Software

- Polybar
- Rofi
- Firefox
- Cava
- VS Code
- Kitty
- Fish Shell
- Dunst
- NeoVim
